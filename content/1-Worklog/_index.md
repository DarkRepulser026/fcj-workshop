**Week 1:**   [Getting familiar with the AWS Free Tier account, AWS CLI, and cost management basics](1.1-week1/)

**Week 2:**   [VPC security controls and core network infrastructure deployment](1.2-week2/)

**Week 3:**   [Core Amazon EC2 concepts and application access management](1.3-week3/)

**Week 4:**   [Amazon S3, Amazon RDS, Auto Scaling/Load Balancing, and Amazon Lightsail](1.4-week4/)

**Week 5:**   [Monitoring with Amazon CloudWatch, Hybrid DNS, and AWS CLI proficiency](1.5-week5/)

**Week 6:**   [Amazon DynamoDB and Amazon ElastiCache for Redis](1.6-week6/)

**Week 7:**   [Advanced networking and hybrid cloud connectivity](1.7-week7/)

**Week 8:**   [Amazon CloudFront and planning the ReviewSentinel capstone project](1.8-week8/)

**Week 9:**   [Serverless architecture with AWS Lambda and API Gateway](1.9-week9/)

**Week 10:**   [AWS SAM, Amazon Cognito, SSL setup, and order processing with SQS/SNS](1.10-week10/)

**Week 11:**   [ReviewSentinel architecture, security, and data model planning](1.10-week11/)
