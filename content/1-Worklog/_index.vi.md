**Tuần 2:** [Kiểm soát bảo mật VPC và triển khai hạ tầng mạng cốt lõi](1.2-week2/)

**Tuần 3:** [Kiến thức cốt lõi về Amazon EC2 và quản lý quyền truy cập ứng dụng](1.3-week3/)

**Tuần 4:** [Amazon S3, Amazon RDS, Auto Scaling/Load Balancing và Amazon Lightsail](1.4-week4/)

**Tuần 5:** [Giám sát với Amazon CloudWatch, Hybrid DNS và thành thạo AWS CLI](1.5-week5/)

**Tuần 6:** [Amazon DynamoDB và Amazon ElastiCache for Redis](1.6-week6/)

**Tuần 7:** [Mạng nâng cao và kết nối hybrid cloud](1.7-week7/)

**Tuần 8:** [Amazon CloudFront và lên kế hoạch cho capstone project ReviewSentinel](1.8-week8/)

**Tuần 9:** [Kiến trúc Serverless với AWS Lambda và API Gateway](1.9-week9/)

**Tuần 10:** [AWS SAM, Amazon Cognito, thiết lập SSL và xử lý đơn hàng với SQS/SNS](1.10-week10/)
