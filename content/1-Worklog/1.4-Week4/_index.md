# Week 4 Achievements

## Development Environment (Cloud9)
- Got familiar with **AWS Cloud9 IDE** — a browser-based integrated development environment with multi-language support, a runtime debugger, and a built-in terminal.

## Object Storage (Amazon S3)
- Mastered the core concepts: Buckets, Objects, and storage classes (Standard, Intelligent-Tiering, Standard-IA, Glacier, Glacier Deep Archive).
- Understood security mechanisms (SSE-S3/SSE-KMS/SSE-C encryption, IAM/bucket policies, ACLs) and management features (Versioning, CRR, Lifecycle, Storage Lens).
- Studied common use cases: Data Lakes, Backup & Restore, Static Website Hosting, Disaster Recovery.

## Relational Databases (Amazon RDS)
- Gained a clear understanding of the supported database engines and storage options (gp2/gp3, io1/io2).
- Understood HA/DR strategies: Multi-AZ, Read Replicas, Global Databases.
- Studied security mechanisms, storage scalability, and backup/recovery strategies (PITR up to 35 days).

## Auto Scaling & Load Balancing
- Mastered scaling strategies: Manual, Dynamic (Target Tracking/Step/Simple), Scheduled, Predictive.
- Understood the role of Launch Templates, Elastic Load Balancing, and Target Groups in scalable architectures.

## Cost Optimization with Lightsail
- Trial-deployed three open-source applications (WordPress, PrestaShop, Akaunting) using Lightsail Blueprints.
- Configured security, backups, vertical scaling, and monitoring alerts for each application.
- Learned about Lightsail Containers for running Docker-packaged applications.
