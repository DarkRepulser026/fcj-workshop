# Week 3 Achievements

## Core Amazon EC2 Knowledge
- Gained a clear understanding of EC2 instance types (Compute, Memory, General Purpose) and their appropriate use cases.
- Understood **Amazon Machine Images (AMIs)**: AWS-provided, Marketplace, and custom AMIs.
- Distinguished between storage options: **EBS volumes** (persistent) and **Instance Store volumes** (ephemeral).

## Security & Advanced Features
- Studied security mechanisms: Key pairs, Security Groups, Network ACLs, and IAM roles.
- Explored key features: Auto Scaling, Elastic Load Balancing, VPC networking, Elastic IPs, and Placement Groups.
- Researched high-availability strategies: Multi-AZ, Local Zones, Outposts, and Wavelength Zones.
- Learned about monitoring tools: Amazon CloudWatch, AWS CloudTrail, and AWS Systems Manager.

## Application Access Management
- Compared granting access via **Access Key/Secret Access Key** and understood the security risks of this approach.
- Understood the benefits of granting access via an **IAM Role** attached to EC2 — the method recommended by AWS.
